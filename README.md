# ⚡ Projeto: CasaVolt Landing Page

Este projeto consiste na criação de uma landing page moderna para serviços elétricos residenciais, inspirada em um design visual de referência. A proposta foi transformar um layout conceitual em uma interface funcional utilizando HTML e CSS, com foco em responsividade, estética e organização de código.

## 🎯 Objetivo

Desenvolver uma landing page profissional para uma empresa fictícia de serviços elétricos, aplicando boas práticas de front-end, estrutura semântica e design responsivo.

## 🖥️ Sobre o Projeto

A CasaVolt é uma landing page voltada para apresentação de serviços elétricos, com destaque para instalações, iluminação inteligente e projetos personalizados.
O layout segue um estilo moderno com gradientes, elementos visuais bem distribuídos e foco em conversão (CTA – Call to Action).

## 📘 Funcionalidades

O projeto inclui os seguintes elementos:

- Header com logo e menu de navegação com dropdown
- Botão de ação “Orçamento”
- Seção principal (hero) com chamada de impacto
- Botões de ação:
  - Ver catálogo
  - Falar com especialista
- Seção de serviços com cards informativos
- Destaque para:
  - Instalações elétricas
  - Iluminação inteligente
  - Atendimento rápido
  - Projetos personalizados
- Seção de cards em destaque (featured)
- Layout totalmente responsivo
- Rodapé com créditos

## 🏷️ Tecnologias Utilizadas

### HTML
- Estrutura semântica: <header>, <main>, <section>, <nav>, <footer>
- Uso de listas para menus e navegação
- Imagens com atributos alt para acessibilidade
### CSS
- Flexbox para layout e alinhamento
- Gradientes para estética moderna
- Efeitos de hover e transições
- Responsividade com media queries
- Separação de estilos:
  - style.css (estilos principais)
  - responsive.css (responsividade)

## 🧠 Aprendizados

Durante o desenvolvimento, foram praticados:

- Estruturação de layout a partir de um design visual
- Criação de menus com dropdown usando apenas CSS
- Uso de Flexbox para organização de componentes
- Implementação de design responsivo
- Trabalhar com gradientes e efeitos visuais modernos
- Organização de arquivos e separação de estilos
- Boas práticas de semântica HTML
